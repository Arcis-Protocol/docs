# Arcis Protocol — Brand Kit

## Brand Identity

**Name:** Arcis (Latin: *of the citadel, of the fortress*)
**Tagline:** The citadel of agent capital
**Voice:** Terse. Declarative. Imperial. Speaks as an institution, not a startup. Latin ornamentation used sparingly — it earns authority, not decoration.

---

## Color System

| Token | Hex | Usage |
|---|---|---|
| Obsidian | `#08080A` | Primary background |
| Stone | `#0F0F12` | Card/surface background |
| Marble | `#16161A` | Elevated surfaces |
| Gold | `#D4AF69` | Primary accent, CTAs, data highlights |
| Gold Bright | `#E2C47E` | Hover states, emphasis |
| Ivory | `#E8E0D0` | Body text |
| Ivory Muted | `#8A8478` | Secondary text, captions |
| Border | `rgba(212,175,105,0.08)` | Subtle dividers |
| Border Hover | `rgba(212,175,105,0.18)` | Active dividers |

**Rule:** Never use pure white. Never use pure black. The palette lives in the space between obsidian and aged parchment.

---

## Typography

| Role | Typeface | Weight | Usage |
|---|---|---|---|
| Display | Cormorant Garamond | 300–600 | Headlines, hero text, stat values |
| Body | Instrument Sans | 400–600 | Paragraphs, UI labels, navigation |
| Mono | SF Mono / Fira Code | 400 | Code, addresses, data |

**Display scale:** Hero 120px → Section 60px → Card 28px → Label 12px
**Body:** 16px/1.6 for paragraphs, 13px for UI, 11px for captions

## Logo Concept

The Arcis logo is an abstract citadel arch — a single Roman arch form suggesting a vault entrance, rendered in gold on obsidian. Minimal, geometric, timeless. Not a shield. Not a coin. An architectural threshold.

## Brand Voice Examples

**Do:**
- "Deposit. Earn. Withdraw. Three functions."
- "Capital infrastructure forged for autonomous economic actors."
- "The ATI is an open standard. The citadel has no gatekeepers."
- "Tres Functiones. Unum Foedus." (Three Functions. One Covenant.)

**Don't:**
- "We're excited to announce..."
- "Our cutting-edge AI-powered DeFi solution..."
- "Join the revolution!"
- Long explanatory sentences. If it needs explaining, simplify.

**Latin usage:** One phrase per section maximum. It should feel like an inscription on stone, not a vocabulary exercise.
